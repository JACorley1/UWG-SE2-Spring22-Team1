Context: User has loaded into the application and is at the home page
Primary Flow:
1.	The user selects the “New” button
2.	A pop-up add page will appear before the user
3.	The user enters in the Name of the habit they want to handle in the Name section
4.	The user selects one of the available buttons for how often they want to update their habit
5.	The user selects the “Add” button
6.	The user is returned to the home page with the habit added successfully
Alternative Flow:
	User Selects Cancel:	
       	3/4/5a. The user selects the “Cancel” button
       	4/5/6a. The user is returned to the home page
	User doesn’t enter any information:
		3b. The user selects the “Add” button
		4b. A prompt appears telling the user that they must fill in all required zones
		5b. The user is returned to the home page.
